spooni_guarma works Standalone.  

To install spooni_guarma:
- Download the resource
  - On [Cfx.re Portal](https://portal.cfx.re/)
- Drag and drop the resource into your resources folder
  - `spooni_guarma`
- Add this ensure in your server.cfg
